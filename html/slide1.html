Псевдоклассы состояния

  • :link - непосещенная ссылка
  • :visited - посещенная ссылка (это означает, что по ссылке уже выполнялся переход, то есть URL-адрес сохранен в журнале истории браузера)

    Из-за причин приватности, браузеры строго ограничивают стили для :visited позволяя изменять только цвета:

    • color
    • background-color
    • border-color - все вместе или для любой из 4-х сторон
    • outline-color
    • column-rule-color
    • fill
    • stroke

    Визуально цвет изменится, но браузер будет указывать не реальный цвет, а цвет ДО ПОСЕЩЕНИЯ ссылки.

    LINK
  • :focus - выделен либо мышью, либо клавиатурой (переход tab-ом в поле input )
  • :hover - ненажатая ссылка (над которой находится указатель мыши)
  • :active - нажатая ссылка (удерживаемая кликом мыши)
