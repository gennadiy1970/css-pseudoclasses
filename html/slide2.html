:nth-of-type(An + B)

Правила :nth-(An + B)

  • правило :nth- ведет отсчет элементов начиная с 1 и не более числа элементов в списке
  • A, n, B - любые целые положительные числа
  • n начинает отсчет с 0
  • ( An + B) будет применяться, если результат 1 и не более числа элементов в списке
  • ( An + B) => ( 0*n + 1) => ( 1) - то же что и first-
  • ( An + B) => ( 2*n + 0) => ( n=0 -> 0, n=1 -> 2, n=2 -> 4, ...) - то же что и even
  • ( An + B) => ( 2*n + 1) => ( n=0 -> 1, n=1 -> 3, n=2 -> 5, ...) - то же что и odd
  • ( -An + B) => ( -1*n + 3) =>
    ( n=0 -> 3, n=1 -> 2, n=2 -> 1) - выделяем первые 3 элемента
  • ( An + B) => ( 1*n + 4) =>
    ( n=0 -> 4, n=1 -> 5, n=2 -> 6, ...) - выделяем все, после первых 3-х элементов
